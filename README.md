Project Overview :

This project provides an automated end-to-end pipeline for the identification and classification of malignant blood cells from microscopic smears. By integrating morphology-based image processing with Machine Learning, the system reduces diagnostic subjectivity and enhances computational efficiency in hematology.


Key Features :

1. Image Preprocessing: Noise reduction and contrast enhancement of microscopic blood smear images using OpenCV.

2. Segmentation: Implementation of Morphological Operations and Superpixel Segmentation to isolate cellular structures.

3. Feature Extraction: Quantitative analysis using Gray-Level Co-occurrence Matrix (GLCM) to capture textural and structural biomarkers.

4. Classification: Machine Learning-based detection of malignant vs. benign cells using Scikit-learn.


Tech Stack :

1. Language: Python

2. Computer Vision: OpenCV

3. Machine Learning: Scikit-learn, NumPy, Pandas
